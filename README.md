# Speech Disorder Classification Using Hybrid Deep Learning

This project classifies dysarthric and non-dysarthric speech using a Hybrid CNN-LSTM-GRU deep learning model. Audio (.wav) files are converted into Mel-spectrograms and processed to extract meaningful speech features for accurate classification.

## Features
- Audio preprocessing using Librosa
- Mel-spectrogram generation
- Hybrid CNN-LSTM-GRU architecture
- Model training and evaluation
- Performance visualization

## Technologies Used
- Python
- TensorFlow / Keras
- Librosa
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Results
The proposed hybrid model achieved high classification accuracy for speech disorder detection and demonstrates the effectiveness of combining CNN, LSTM, and GRU networks for speech analysis.
